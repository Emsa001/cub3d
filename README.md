# Cub3D 
Cub3D is a 3D game from 42 school projects...
## Install & Run

for 'C' and 'Make'
```bash
sudo apt install build-essential -y
```
for mlx library
```bash
sudo apt install libx11-dev libxext-dev libxft-dev libxrandr-dev libbsd-dev -y
```
run game with all its features
```bash
make runb
```
